Descrição

    Navbar: A barra de navegação é fixa no topo e colapsa em um botão de menu no modo mobile.
    Main Header: Uma seção de cabeçalho principal com fundo azul e texto centralizado.
    Content Section: Uma seção de conteúdo com cards que se reorganizam conforme o tamanho da tela. No exemplo, há um card com imagem, título, texto e botão.
    Footer: Um rodapé simples com texto centralizado.

Este exemplo usa o Bootstrap para garantir que a página seja responsiva. 
O layout se ajustará automaticamente em diferentes tamanhos de tela, 
mostrando uma navegação adequada e reorganizando os elementos conforme necessário.
